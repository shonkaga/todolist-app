## Running on Windows:
1. open powershell 

2. run the following commands 
    ```shell 
        cd todolist-app
        python -m venv venv
        .\venv\Scripts\Activate.ps1
    ```

    if you get an error when trying to activate then open powershell as an admin and run
    `Set-ExecutionPolicy Unrestricted -Force`
